# Inventra - Medical Inventory Management System

A RESTful API for managing medical inventory.

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Postman

## Features
- Add medicines
- View all medicines
- View a medicine by ID
- Update medicine quantity
- Delete medicines

## API Endpoints
- GET /medicines
- GET /medicines/:id
- POST /medicines
- PUT /medicines/:id
- DELETE /medicines/:id
